What HTTPLint Catches

90 checks across 6 categories, covering every aspect of HTTP client and server configuration.

HC
📡

HTTP Client

Detects missing error handling on fetch/axios calls, no response status checks, ignoring HTTP errors silently, unhandled promise rejections, and unsafe response parsing.

15 checks
HH
🛡️

Headers & CORS

Catches missing security headers (CSP, HSTS, X-Frame-Options), permissive CORS with wildcard origins, missing Content-Type headers, and absent X-Content-Type-Options.

15 checks
CM
🔌

Connection Management

Finds missing connection pooling, connection leaks from unclosed sockets, absent keep-alive configuration, TCP connections never properly closed, and no connection timeout set.

15 checks
CC
💾

Caching

Detects missing Cache-Control headers, absent stale-while-revalidate directives, aggressive caching of dynamic content, no ETag or Last-Modified headers, and stale cache invalidation.

15 checks
RP
📊

Request/Response Patterns

Catches unbounded response bodies, no request size limits, missing pagination on list endpoints, N+1 HTTP call patterns, and responses without compression enabled.

15 checks
EM
🔍

Error & Monitoring

Detects swallowed HTTP errors, missing status code logging, absent request tracing headers, no latency metrics collection, and unmonitored upstream dependencies.

15 checks
